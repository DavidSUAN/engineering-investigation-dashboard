# TMC空客组工程调查看板

纯前端工程调查可视化看板，用于实时监控TMC空客组工程调查项目进度。

## 快速开始

1. 用 Chrome / Edge 打开 `index.html`
2. 上传仓库中的 `示例数据_工程调查跟踪单.xlsx`（520 条虚构数据，可用于体验全部功能）
3. 选择文件夹或拖拽 Excel 文件上传（**只需操作一次，后续自动加载**）
4. 看板将自动展示并保存数据，下次打开直接进入看板

## 功能特性

### 1. 一次性上传，自动恢复
- 首次上传后数据自动缓存至浏览器（IndexedDB）
- 文件夹模式下自动保存文件句柄，下次打开自动读取最新文件
- 拖拽上传模式缓存解析好的数据，无需重新上传
- 上传页面提示"只需上传一次，后续打开自动加载"
- 顶部保留「刷新数据」和「重新上传」按钮

### 2. 顶部统计卡片
五组关键数字：总计、进行中、超期、注意、正常。

- **悬停交互**：鼠标移到超期/注意/正常卡片上弹出浮层，列出该分类下所有调查的故障描述、负责工程师、已过天数。点击任意行跳转详情。
- **点击卡片**：跳转到页面下方对应颜色分区。

### 3. 三色分区
根据已过天数与调查天数的比值将项目分为三个区间：

| 区域 | 颜色 | 条件 |
|------|------|------|
| 超期 | 红色 | 已过天数 ≥ 调查天数 |
| 注意 | 橙色 | 调查天数×2/3 < 已过天数 < 调查天数 |
| 正常 | 绿色 | 已过天数 ≤ 调查天数×2/3 |

### 4. 工程调查卡片
每条卡片显示：
- 飞机注册号、机型、ATA章节、执管基地
- **状态标签**（蓝色系渐变）：调查中 → 科室讨论 → WG周会 → 可靠性周会
- 待办事项标签（紫色）、可靠性标签（绿色）、故障原因标签（天蓝色）
- 右上角显示**"剩余X天，已过Y天"**，颜色按紧急程度标识

### 5. 统计图表（可折叠）
ATA章节统计和月度趋势图折叠在「统计图表」区域，默认收起，点击展开。

**ATA章节统计**：一行三列（总览 / A350 / A330），柱状图展示各章节调查数量，点击柱子查看故障原因饼图，点击扇区快速筛选。

**月度趋势**：一行三列折线图，支持左右翻页浏览历史月份，点击数据点跳转到对应月份。

### 6. 工程师工程调查统计
堆叠柱状图展示各工程师的调查分布（超期/注意/正常/已完成），悬停提示"点击查看该工程师的月度趋势图"。点击柱子弹出月度趋势弹窗，**不影响主看板筛选条件**。

### 7. 筛选条件
- 支持故障类型、机型、注册号（可搜索）、基地、ATA章节、工程师、故障原因、待办事项等维度
- 多选，实时联动所有图表和卡片
- 每项可独立重置
- 日期范围筛选

### 8. 科室成员管理
- 「仅显示科室成员」开关（默认开启）
- 「管理」按钮多选成员名单
- 保存至浏览器 localStorage，跨会话持久化

### 9. 待办事项看板
筛选"待办事项 → 有"后出现「待办看板」按钮，弹窗展示 WG 跟踪事项的表格，可按工程师筛选，点击行跳转详情。

### 10. 详情页
- 顶部卡片：飞机注册号、机型、基地、ATA章节、**工程师姓名**
- 流程进度条：调查中 → 科室讨论 → WG周会 → 可靠性周会
- 详细信息：基本信息、故障信息、调查信息、会议记录、关键字

### 11. 使用指南
标题旁「使用指南」按钮，弹窗详细说明所有功能的使用方法。

## 使用方法

### 加载数据
**方式一（推荐，Chrome/Edge）：** 选择文件夹 → 点击文件名加载。后续打开自动恢复。

**方式二：** 拖拽 .xlsx 到上传区。

### 筛选与查看
展开筛选面板设置条件 → 查看图表和卡片 → 点击卡片进详情。

### 快捷操作
- `Esc` 关闭所有弹窗
- 点击弹窗外部遮罩也可关闭

## 数据格式要求

Excel 工作表"调查项目清单"，按表头名称自动匹配列，不依赖列顺序。

## 状态判断逻辑

按优先级从高到低：
1. 有可靠性周会日期 → **可靠性周会**
2. 有 WG 会议决议 → **WG周会**
3. 有科室讨论 → **科室讨论**
4. 否则 → **调查中**

## 颜色分区逻辑

```
threshold = ceil(调查天数 × 2/3)
已过天数 ≥ 调查天数 → 超期（红）
已过天数 > threshold → 注意（橙）
已过天数 ≤ threshold → 正常（绿）
```

调查天数无效或已填结束日期 → 归类为"已完成数据"。

## 技术栈

- HTML + JavaScript（无框架）
- Tailwind CSS (CDN)
- SheetJS 解析 Excel
- Chart.js 图表
- IndexedDB 持久化存储
- 浏览器直接打开即可

## 数据安全

所有数据在浏览器本地处理，不上传至任何服务器。
