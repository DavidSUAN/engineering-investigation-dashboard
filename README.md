# TMC空客组工程调查看板

一个纯前端的工程调查可视化看板，用于实时监控TMC空客组的工程调查项目进度。

## 功能特性

### 1. 总览统计
- **进行中**: 显示所有未完成的工程调查总数
- **超期**: 显示超过12天的调查项目数量（红色标识）
- **即将超期**: 显示10-12天的调查项目数量（橙色标识）
- **正常**: 显示9天以内的调查项目数量（绿色标识）

### 2. 四色时间分区
根据调查开始日期与当前日期的间隔，将项目分为四个颜色区间：
- 🔴 **红色 (>9天)**: 时间较长，需重点关注
- 🟠 **橙色 (6-8天)**: 时间中等，需关注
- 🟡 **黄色 (3-5天)**: 时间较短
- 🟢 **绿色 (0-2天)**: 刚开始，时间充裕

### 3. 多维度筛选
支持按以下维度筛选：
- 故障类型 (source)
- 机型 (fleet)
- 飞机注册号 (aircraft registration)
- 执管基地 (base)
- ATA章节 (ATA chapter)
- TMC工程师 (TMC engineer)
- 开始日期范围 (start date range)

### 4. 详情查看
点击任意项目卡片可查看完整信息：
- 基本信息（飞机注册号、机型、基地、日期等）
- 调查流程进度可视化（调查中 → 科室讨论 → WG周会 → 可靠性周会）
- 故障信息（ATA章节、故障描述）
- 调查信息（工程师、报告标题、评分）
- 会议记录（科室讨论、WG决议、可靠性周会等）
- 关键字信息（ECAM警告、故障信息、故障源）

### 5. 数据导出
支持将当前筛选结果导出为Excel文件。

## 使用方法

### 1. 打开看板
直接在浏览器中打开 `index.html` 文件即可。

**支持的浏览器**:
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### 2. 上传Excel文件
- 拖拽Excel文件到上传区域，或点击选择文件
- 支持 `.xlsx` 格式
- 文件必须包含名为"调查项目清单"的工作表

### 3. 查看总览
上传成功后自动跳转到总览页面：
- 顶部统计卡片显示各类项目数量
- 下方按四色分区展示项目列表
- 每个分区可点击展开/折叠

### 4. 使用筛选
- 点击"筛选条件"展开筛选面板
- 在任意筛选项下拉框中选择条件（支持多选）
- 可使用日期范围筛选
- 点击"重置筛选"清空所有条件

### 5. 查看详情
- 点击任意项目卡片进入详情页
- 查看完整的调查信息和流程进度
- 点击"返回总览"回到列表页

### 6. 导出数据
- 点击页面底部的"导出筛选结果"按钮
- 自动下载当前筛选结果的Excel文件
- 文件名格式: `工程调查看板_筛选结果_YYYYMMDD_HHmmss.xlsx`

Excel文件必须包含名为"调查项目清单"的工作表，列结构如下：

| D | source_other | OTHER备注 |

| G | base | 执管基地 |
| H | fault_date | 故障日期 |

| L | tmc_engineer | TMC工程师 |

| O | end_date | 调查结束日期 |
| P | count | 计数 |

## 状态判断逻辑

系统会根据以下规则自动判断调查状态（优先级从高到低）：
1. 如果"可靠性周会日期"列有值 → 状态为"可靠性周会"
2. 否则如果"WG会议决议"列有值 → 状态为"WG周会"
3. 否则如果"科室讨论日期/结果"列有值 → 状态为"科室讨论"
4. 否则 → 状态为"调查中"

## 超期判断逻辑

基于调查开始日期计算：
- **超期**: 超过12天（红色标识）
- **即将超期**: 10-12天（橙色标识）
- **正常**: 9天以内（绿色标识）

## 技术栈

- **前端框架**: 纯HTML + JavaScript（无需后端）
- **UI框架**: Tailwind CSS (CDN)
- **Excel解析**: SheetJS (xlsx.js)
- **部署**: 单文件，直接打开即可使用

## 数据安全

- 所有数据处理在浏览器本地完成
- 不会上传任何数据到服务器
- 不收集用户信息

## 跨平台支持

本看板完全跨平台，支持：
- Windows (Edge, Chrome, Firefox)
- macOS (Safari, Chrome, Firefox)
- Linux (Chrome, Firefox)

只需浏览器即可使用，无需安装任何额外软件。

## 故障排除

### 问题：无法上传文件
- 确认文件格式为 `.xlsx`
- 确认文件包含"调查项目清单"工作表

### 问题：数据解析失败
- 检查Excel文件是否损坏
- 确认列结构符合要求
- 尝试在Excel中重新保存文件

### 问题：筛选无结果
- 检查筛选条件是否过于严格
- 点击"重置筛选"清空条件
- 确认数据中存在符合条件的记录

### 问题：导出失败
- 确认浏览器允许下载文件
- 检查是否有弹窗拦截
- 尝试使用Chrome或Edge浏览器
